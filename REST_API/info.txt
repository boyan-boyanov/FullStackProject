1. Първо инициализираме приложението с npm init -y в папката REST_API
2. Инсталираме npm i express
3. Създаваме index.js
4. Инсталиране на nodemon
5. Стартираме с nodemon index.js
6. В папката която ще се къмитва се натиска ctrl + shft + p после се пише Publish to GitHub и се избира Private или Public
    преди това се прави README.md, ще поиска message за първия къмит.
